## Installation
Add this line to your application's Gemfile:

```ruby
gem 'wordpress'
```

And then execute:
```bash
$ bundle install 
$ rails g active_admin:install
$ rails g wordpress:install
$ rake db:migrate
$ rake db:seed
```

Or install it yourself as:
```bash
$ gem install wordpress
```
